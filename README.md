# TryWindowsBatch

主にWindows環境で動作するプログラムの練習用プロジェクトです。

## Windowsコマンドプロンプトの使い方

コマンド履歴
- [↑][↓]
 - 履歴を表示する。
 - [F3]で直前の履歴を表示。[↑]で代用可能？
- コマンド入力途中に[F8]
 - 入力途中の内容に一致する履歴を表示する。
- [F7]
 - 履歴一覧を表示する。[↑][↓]で選択し、[Enter]で実行。[←][→]で再編集。
